Working With Pixels:


Affiliate Buddies has a special feature to notify affiliates about each new account or FTD made by their referral.

In order to use that feature correctly just follow those instructions:

  1. Goto creative materials click edit near the relevant banner and get to get tracking code page
  2. Put 123456 on the dynamic parameter field and then “get code”
  3. Save the generated direct link you’ve got on the side.
  4. Goto top menu:  “Affiliation Management” -> “affiliate list”
  5. Search for the affiliate you want to see that pixel
  6. Click edit on the left
  7. Scroll down till you see “Pixel monitor” tab, if it collapsed then click on that and you’ll get the menu visible.
  8. Put the server to server pixel in the pixel code field.
  1. If you want to shave some information with the affiliate then use the variables below, for example if the affiliate want to know the trader id then this is how url should look like:
    http://www.abc.com?trader_id={trader_id}
    if the affiliate wants to get the exact parameter coming from his end then it should look like:

http://www.abc.com?param={dynamic_parameter}

  1. Choose the fire trigger method (account / sale ..) and save.
  2. Now the testing, take the affiliated url (aka #3), tell the affiliate to replace 123456 with his parameter on any referral made by him.
  3. Once affiliate made a test using affiliated link, then open an account, our system will collect the data using the API integration and trader/player report of the affiliate program will show the value of the dynamic parameter under the title “param” on the same row of this registration.



  • If you can’t see any value in the param field on the trader/player report then no pixel will fire in case it needs a dynamic parameter.
  • Make sure the dynamic parameter coming is not having spaces or dashes (-) in it.
  • Pixels are getting fire once XX minutes according to the settings made by the affiliate manager on the integration, so affiliates can’t expect to get the pixel fire a second right after registration or the deposit.
