Adding marketing Tools

Click On "Marketing tools -> Add New

Once you did that please choose LANGUAGE -> Insert your refer page -> Choose promotion ( Not must ) -> insert creative name -> and alt name ( Good for SEO )  

Now you are ready to upload your creative all you need to do is to click on file and to import your creative with one click and to save


3.2 How to create new promotion

Go to marketing tools click on add promotion

  • Choose your new Promotion name
  • Go to promotion list and activate your new Promotion
  • how to create a special promotion for specific affiliate ( just add the number of the affiliate ID )
